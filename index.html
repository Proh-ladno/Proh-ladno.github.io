Описание

Мечурская рыба - это редкий предмет, который можно получить, ловя рыбу в Месть-Мозгов. Используется в качестве крафтового ингредиента для создания некоторых предметов.

Крафт

Мечурская рыба не крафтится.

Использование

Мечурская рыба используется в качестве крафтового ингредиента для создания:

  • Рыбов кираса
  • Уазаруку

Весь материал, представленный на данном сайте, является собственностью игры Террария.
